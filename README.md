# go-kalshi

Experimental Go wrapper for Kalshi API

## Instructions

### Build test program

```
git clone <this repo>
cd go-kalshi
make
```

### Run test program

Create a `.env` file that looks like this:

```
KALSHI_USERNAME="your kalshi login (email)"
KALSHI_PASSWORD="your kalshi password"
```

Now run the test program:

```
./run.sh
```
